Adding a hosted new demo

  1. Setup an EC2 instance for demo purposes
  2. Ensure Java and Maven is installed
  3. Git clone the project from the repo
  4. export MAVEN_OPTS='-Xmx1500m -XX:+UseConcMarkSweepGC'
  5. mvn jetty:run
  6. Launch browser to your instance:8080/index.jsp
  7. Update this index page to include your example
